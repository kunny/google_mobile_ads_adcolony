# AdColony mediation plugin for Google Mobile Ads Flutter

Google Mobile Ads Flutter mediation plugin for AdColony.

## Use this package as a library

```yaml
dependencies:
  google_mobile_ads_adcolony:
    git:
      url: https://github.com/kunny/google_mobile_ads_adcolony.git
      ref: <tag>
```

For example, if you want to depend on version `2.4.0`:

```yaml
dependencies:
  google_mobile_ads_adcolony:
    git:
      url: https://github.com/kunny/google_mobile_ads_adcolony.git
      ref: 2.4.0
```

Refer to the available tags on [here](https://github.com/kunny/google_mobile_ads_adcolony/tags).

## Additional code required

No additional code is required for AdColony integration.

This is not an official Google product.
